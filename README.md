# Library Management System

Это веб-приложение для управления заказами книг в библиотеке, с разделёнными ролями пользователей: обычные пользователи и библиотекари.

## Функционал

- **Регистрация и Авторизация**
  - Пользователи могут зарегистрироваться и войти в систему.
  - Реализована проверка ролей и доступ к страницам.

- **Профиль пользователя**
  - Просмотр и редактирование профиля.

- **Основная страница**
  - Отображение информации о приложении и приветствие.

- **Кабинет библиотекаря**
  - Навигация по разделам: Главная, Открытые заявки.
  - Просмотр и обработка открытых заявок на книги: подтверждение или отклонение заказа.
  - Современный дизайн с использованием Ant Design.

- **Обработка ошибок**
  - Кастомная страница 404 для несуществующих маршрутов.
  - Кастомная страница 403 для уведомления пользователя о недостатке прав

## Технологии

- React + TypeScript
- React Router DOM
- Ant Design
- Контекст авторизации и провайдеры данных
- Модульная структура компонентов

## Быстрый старт

1. Установите зависимости:
   ```bash
   pnpm install
   ```

2. Запустите проект, в корне выполните:
   ```bash
   pnpm dev
   ```

3. Откройте приложение в браузере по адресу `http://localhost:5173`.

## Структура проекта

```
frontend/
├── src/
│   ├── app/          # Главная архитектура приложения и маршрутизация
│   ├── pages/        # Страницы: Home, Profile, Login, Register, NotFound
│   ├── widgets/      # Виджеты для страниц, например, LibrarianDashboardTab
│   ├── entities/     # Типы и сущности пользователя и т.д.
│   └── shared/       # Переиспользуемые компоненты и константы
```

## Роли пользователей

- **User** — обычный пользователь библиотеки.
- **Librarian** — библиотекарь, имеет доступ к панели администрирования.

## Для разработчиков

- Все основные маршруты настраиваются в `frontend/src/app/Router.tsx`.
